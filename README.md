# GrowthX Backend

Production-style FastAPI backend for the GrowthX productivity app.

## Architecture

Flutter → FastAPI → PostgreSQL (SQLAlchemy + Alembic) → JWT Auth → AI Layer (LangChain/RAG, future)

## Folder Structure

```
app/
  main.py         # App assembly only — no business logic
  config.py       # Single source of truth for settings (env vars)
  database.py     # SQLAlchemy engine/session, get_db() dependency
  routers/        # HTTP layer — request in, response out
  models/         # SQLAlchemy ORM models (DB shape)
  schemas/        # Pydantic models (API contract / validation)
  services/       # Business logic
  repositories/   # DB query layer
  core/           # Security, JWT, shared dependencies
  utils/          # Small stateless helpers
```

## Local Setup

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate      # Windows: venv\Scripts\activate
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Copy `.env` and adjust values if needed (already included with dev defaults).

4. Run the server:
   ```bash
   uvicorn app.main:app --reload
   ```

5. Verify it's alive:
   - http://127.0.0.1:8000/health
   - http://127.0.0.1:8000/docs  (interactive Swagger UI, auto-generated by FastAPI)
